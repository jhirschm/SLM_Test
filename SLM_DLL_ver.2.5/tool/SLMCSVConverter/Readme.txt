What is it?
==========

The SLMCSVConveter is a utility to convert from csv to bmp.  
The converted BMP is created in the same folder as the CSV file.
CSV file must be in SLM format.


Files
=====

Readme.txt:				This file
SLMCSVConverter.exe:	Program file
test.bat				Sample batch file

How to Run
==========

SLMCSVConveter.exe c:\temp\test.csv

Requirements
==========
?.NET Framework 4 or later
